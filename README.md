 Traffic Signal Timer Project
 Overview:
The Traffic Signal Timer Project is an automated traffic control system that manages signal lights using predefined time intervals. The goal of this project is to ensure smooth traffic flow, reduce congestion at intersections, and improve overall road safety through systematic signal operation.
 Objectives:
Automate traffic signal control using a timer-based system
Reduce traffic congestion and waiting time
Improve traffic discipline and road safety
Demonstrate practical application of electronics/embedded systems
 Working Principle:
